Отрисовывает переданные вершины.

Parameters:
Name Type Attributes Default Description
verts Float32Array <nullable>

Вершины, которые надо отрисовать. Если не передано, будут использоваться вершины из стандартного буфера (v6.RendererGL#buffers.default).
count number

Количество вершин, например: 3 для треугольника.
mode constant <optional>
 STATIC_DRAW

Например: GL_STATIC_DRAW.
_sx number <nullable>

X scale. Used for backwards-compability with v6.Renderer2D#vertices.
_sy number <nullable>

Y scale. Used for backwards-compability with v6.Renderer2D#vertices.
